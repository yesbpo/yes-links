## Architecture

```
YesLinksProvider
  token + baseUrl → context
         ↓
    useYesLinks()
         ↓
  createApiClient({ token, baseUrl })   ← useMemo (stable ref)
    ↙              ↘
useLinks          useStats
  getLinks()    getDashboardSummary()
         ↓
  YesLinksDashboard (managed mode)
    no dataSource prop needed
```

## Usage Pattern (Sprint 1 Goal)

```tsx
<YesLinksProvider token={jwt} baseUrl="https://api.yes.com/v1">
  <YesLinksDashboard />   {/* zero dataSource wiring */}
</YesLinksProvider>
```
